# Named-Entity-Recognition-on-News-Articles-
Named Entity Recognition (NER) is a subtask of natural language processing (NLP) that focuses on identifying and classifying named entities in text into predefined categories such as people, organizations, locations etc.The explosion of news articles presents a major hurdle in efficiently extracting and analyzing thecrucial information they hold. Manually sifting through this massive data to pinpoint and categorizeimportant entities like people, organizations, locations, and dates is an extremely  time-consumingand tedious task , prone to human error.
